 SmartResume AI

SmartResume AI helps users improve their CVs using AI.  
Paste your CV text and get professional suggestions, impactful phrases, and improvement tips.

 Features
- AI analysis of CV
- Suggestions for stronger wording
- Simple, responsive interface
- PHP + OpenAI GPT-4o

 Setup
1 Clone repository
2 Place in `htdocs` (for XAMPP)
3 Add your OpenAI API key in `analyze.php`
4 Start Apache, open `http://localhost/smartresume-ai`

Tech Stack
PHP · HTML · CSS · JS · OpenAI GPT-4o
